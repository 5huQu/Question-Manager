# 阶段 4：试题篮 UI

目标：实现常驻、可收起的多试卷试题篮侧栏，并把题库页、题目详情页的“加入试题篮”接入多试卷工作流。

## 进度

- [x] 创建 `QuestionBasket` 组件目录
- [x] 展示多份试卷草稿
- [x] 支持新建试卷
- [x] 支持切换当前试卷
- [x] 支持题目添加到指定试卷
- [x] 支持题目移除
- [x] 支持编辑每题分值
- [x] 支持题目排序
- [x] 实时显示题数和总分
- [x] 支持侧栏收起/展开
- [x] 在题库页接入
- [x] 在题目详情页接入
- [ ] 完成浏览器交互验证

## 验收标准

- 用户能在不离开题库页面的情况下完成基础组卷。
- 多份试卷草稿之间状态互不干扰。
- 移动端和桌面端布局不出现遮挡。
