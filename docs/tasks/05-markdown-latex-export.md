# 阶段 5：Markdown / LaTeX 导出

目标：用 Markdown 和 LaTeX 替代 Word 导出，生成更工整、可继续排版和编译的试卷内容。

## 进度

- [x] 设计 Markdown 导出格式
- [x] 设计 LaTeX 导出格式
- [x] 支持导出学生版
- [x] 支持导出教师版
- [x] 支持标题、副标题、总分、时长
- [x] 支持题号和每题分值
- [x] 支持按题目顺序导出
- [x] 支持按大题分组导出
- [x] 保留数学公式
- [x] 验证 Markdown 预览效果
- [x] 验证 LaTeX 可编译性

## 验收标准

- Markdown 文件可直接预览。
- LaTeX 文件可用于后续编译 PDF。
- 学生版只包含题干。
- 教师版每题题干后接参考答案和解析。
