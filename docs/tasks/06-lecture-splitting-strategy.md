# 讲义版资料切分试验策略

这份文档记录四份讲义型解析版 DOCX 的第一版切题规则。当前目标只是生成可复核的切题预览，并借用现有 PDF 切分复核窗口展示；本轮不做 OCR，也不写入题库。

## 样本

| 文件 | 公式形态 | 试验路线 |
| --- | --- | --- |
| `第04讲 数列的通项公式（复习讲义）（全国通用）（解析版）.docx` | 主要是可编辑 OMML 公式 | 转 PDF 生成复核预览，按 DOCX/PDF 文本标记切题 |
| `专题3.3 导数与函数的极值、最值（举一反三讲义）（全国通用）（解析版）.docx` | 主要是可编辑 OMML 公式 | 转 PDF 生成复核预览，按 DOCX/PDF 文本标记切题 |
| `专题21 解三角形（10大题型）（解析版）.docx` | 图片/OLE 公式较多 | 转 PDF，按视觉 PDF 文本层做 OCR 路线的切题预览 |
| `第04讲 随机变量的数字特征（解析版）.docx` | 图片/OLE 公式和表格较多 | 转 PDF，保留整页宽度切图，避免表格和图形丢失 |

## 哪些内容切成候选题

讲义不能完全按试卷规则切。第一版把内容分成两层识别：

1. 讲解层题目：
   - `【典例...】`、`典例...`
   - `【例...】`、`例...`
   - `【变式...】`、`变式...`
   - `【变式训练...】`
   - `【即学即练...】`

2. 训练层题目：
   进入训练区后，`1．`、`2．`、`3．`、`1.`、`1、`、`1（...）` 这类编号行都视为题目开头。

训练区标题包括：

- `限时训练`
- `课后训练`
- `课后作业`
- `课时作业`
- `过关检测`
- `小试牛刀`
- `巩固提升`
- `巩固练习`
- `课堂检测`
- `随堂检测`
- `随堂练习`
- `当堂检测`
- `核心突破提升练`
- `真题溯源通关练`
- `常考题型过关练`
- `综合检测`
- `达标检测`
- `一、单选题`
- `二、多选题`
- `三、填空题`
- `四、解答题`

很多讲义末尾的训练区不会直接写“训练”，而是切到类似试卷的 `一、单选题 / 二、多选题 / 三、填空题 / 四、解答题`。这些标题也必须视为训练区，否则课后训练和限时训练很容易漏切。

训练区之外，普通 `1．/2．/3．` 不直接切成题，因为它们经常是知识点、步骤、性质或方法说明。作为兜底，如果编号行明显是题干，例如含年份来源、`已知/设/若/如图/求/证明` 等题干特征，也会切成训练候选。

## 哪些内容只做上下文

这些内容只作为上下文标题写入复核备注，不单独切成题：

- `知识点...`
- `考点...`
- `题型...`
- `模块...`
- `方法技巧`
- `考情分析`
- `知识点梳理`
- `思维导图`

复核项会保存最近的章节标题，后续可以作为来源章节、标签建议或知识点上下文。

## 边界规则

候选题从标记行开始，到下一个候选题标记或下一个主要章节标题之前结束。裁图使用整页宽度，优先保住公式、表格、图形和解析。跨页题会把每页裁剪段纵向拼接成一张预览图，直接用于现有复核窗口。

编号题规则特意排除了 `0.1`、`0.06` 这类小数，避免把概率表、分布列表格里的数值误切成“第 0 题”。

整题截图保住图形不等于已经识别图块。本轮会额外调用 PDF 图形簇检测，把函数图、几何图、统计图等写入 `figures_json`。复核窗口右侧会显示图框数量，题图也可以在预览图上单独框选、调整或删除。

分布列、概率分布表、分布表、分布律、列联表这类内容本质上是题干里的结构化文字/表格。即使 Word 转 PDF 后被解析成图片或图形簇，也不作为独立题图写入 `figures_json`，只保留在整题截图中交给 OCR 转录。

图片/OLE 公式较多的 Word 转 PDF 后，选项里的公式或字母也可能变成很小的嵌入图片，例如 `B.\frac{1}{3}`。这类极小 `image` 块属于题干文字/公式，不作为独立题图写入 `figures_json`。

讲义模板里的页眉、页脚、水印或来源 logo 有时会被 PDF 解析成横跨整页的 `graphic`。如果图块贴着页面顶端或底端、宽度接近整页且高度较浅，视为页面品牌装饰，不作为题干图。

## 本轮输出

试验脚本会写入：

- `experiments/pdf_slicer/runs/<runId>/` 下的转换 PDF 和页面渲染图
- `output/lecture_cuts/` 下的讲义候选切图
- `output/lecture_segments.json`
- `pdf_slicer_runs` 和 `pdf_slicer_review_items`
- 每个候选题的 `figures_json`，用于记录独立题图位置

试验脚本不会写入 `question_bank_items`。
